Privacy Policy for Structura

Last Updated: November 2025
Developer: Skyline Bot Studio
Contact: skylinebotstudio@outlook.com

Introduction

Structura is a Discord automation bot developed by Skyline Bot Studio. This Privacy Policy explains what information the bot collects, how it is used, and how your information is protected. By using Structura, you agree to the practices described in this policy.

Information We Collect

Automatically Collected Data

Structura only collects and stores the minimum data needed for the bot to function:

  • User IDs
  • Guild (Server) IDs
  • Channel and Category IDs
  • Server layout data
  • Saved templates
  • Permission settings
  • Premium subscription status
  • Technical error logs (no message content)

Information We Do Not Collect

Structura does not store, log, or access:

  • Message content
  • Direct messages
  • User conversations
  • Attachments
  • Personal profile information
  • IP addresses
  • Sensitive personal data

Structura is intentionally designed to avoid collecting unnecessary or personal information.

How Your Data Is Used

Structura uses stored data solely to:

  • Create server layouts
  • Save and load templates
  • Manage channels and categories automatically
  • Track premium features
  • Maintain bot performance and fix issues

Your data is never used for advertising, profiling, or analytics.

Data Sharing

Structura does not sell, share, or transfer your data to:

  • Advertisers
  • Marketing platforms
  • Data brokers
  • Analytics companies
  • Third parties

The only exception is when required by law (rare).

Data Retention

We retain data only as long as needed:

  • Templates remain saved until deleted
  • Server layout data is deleted when the bot is removed from a server
  • Error logs are temporary and cleared during updates

Data Deletion Requests

You may request deletion of:

  • Server data
  • Templates
  • User data
  • Premium status

Email: skylinebotstudio@outlook.com
We respond within 72 hours.

Children’s Privacy

Structura is not intended for individuals under 13, in compliance with Discord’s Terms of Service.

Policy Changes

This Privacy Policy may be updated at any time. Continued use of Structura means you accept any changes.
