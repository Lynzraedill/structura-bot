Getting Started with Structura

This page provides a simple overview of how to begin using Structura in your Discord server.

1. Invite the Bot

Use this link to invite Structura to your server:

➕ Add Structura to Your Server

Make sure you give the bot the recommended permissions so it can create categories and channels.

2. Basic Usage

Once Structura is in your server, you can start with basic commands such as:

  • Listing available layouts.
  • Applying a layout to create channels and categories.
  • Saving your current server structure as a reusable template.

Refer to your in-Discord help command or documentation for the exact slash command names once they are finalized.

3. Tips for Best Results

  • Test new layouts in a private or test server first.
  • Be sure you understand which channels will be created or modified.
  • Use saved templates to quickly set up similar servers (for clients, projects, or communities).
