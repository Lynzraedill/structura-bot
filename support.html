Support and Contact

If you need help using Structura, have found a bug, or want to request a feature, you can reach out in a few ways.

Before you ask

  • Make sure Structura has the right permissions (Manage Channels, Send Messages, etc.).
  • Check that the bot is online in your server.
  • Try the basic commands again and note any error messages.

Contact Options

  • Email support: skylinebotstudio@outlook.com
  • If provided, use the official support Discord server link or ticket system.

When you contact support, please include:

  • Your Discord username and server name.
  • What command you tried.
  • Screenshots or a copy of any error message.
